Sobre Mi Proyecto

Este proyecto forma parte del módulo 2 de mi Bootcamp en Soy Henry, donde estoy aprendiendo a convertirme en un desarrollador Full Stack. El objetivo principal es crear una página web dinámica enfocada en el mundo del cine, la cual he llamado CinemaHub.

Objetivo del Proyecto

El proyecto CinemaHub tiene como objetivo proporcionar una plataforma interactiva donde los usuarios puedan explorar la cartelera actual, aprender sobre la historia del cine, y descubrir sitios web recomendados relacionados con la industria cinematográfica.

A través de este proyecto, estoy aplicando los conceptos y tecnologías que estoy aprendiendo en el bootcamp para crear una aplicación completa y funcional.

Tecnologías utilizadas

Durante el desarrollo de este proyecto, estoy utilizando una variedad de tecnologías tanto del lado del cliente como del lado del servidor, que incluyen:

HTML, CSS y Bootstrap

Para el diseño y la estructura visual del sitio web.

JavaScript y AJAX

Para la creación de interacciones dinámicas y el manejo de peticiones asincrónicas.

Node.js y Express

Para construir el backend y manejar las peticiones de los usuarios.

MongoDB y Mongoose

Para almacenar y gestionar los datos de las películas.

Git y GitHub

Para el control de versiones y la colaboración en el proyecto. Puedes ver el código fuente de este proyecto en mi repositorio de GitHub.
